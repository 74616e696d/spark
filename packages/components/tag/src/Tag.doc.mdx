# Tag

Tag component is used to trigger an action or event, such as submitting a form, opening a Dialog, canceling an action, or performing a delete operation.

## Install

```sh
npm install @spark-ui/tag
```

## Import

```js
import { Tag } from '@spark-ui/tag'
```
