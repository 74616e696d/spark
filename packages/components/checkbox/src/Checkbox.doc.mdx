# Checkbox

A control that allows the user to toggle between checked and not checked.
It is used in forms when a user needs to select multiple values from several options.

## Install

```sh
npm install @spark-ui/checkbox
```

## Import

```tsx
import { Checkbox } from '@spark-ui/checkbox'
```

## Props

<ArgTypes of={Checkbox} />

## Usage

Checkboxes accept children, which are rendered as the label. Checkboxes should **always have a label**.

<Canvas of={CheckboxStories.Default} />

## Hidden label

In certain cases, a visible label may not be necessary, such as when using a checkbox to select a table row. To achieve this behavior, the `VisuallyHidden` component can be used.

<Canvas of={CheckboxStories.HiddenLabel} />

## Disabled

Use `disabled` prop to indicate the switch is disabled.

<Canvas of={CheckboxStories.Disabled} />

## Uncontrolled

Checkboxes are not selected by default. The `defaultChecked` prop can be used to set the default state. Optionally `onCheckedChange` prop can be used to get the current state of the checkbox.

<Canvas of={CheckboxStories.UncontrolledState} />

## Controlled

Use `checked` and `onCheckedChange` props to control the state of the checkbox.

<Canvas of={CheckboxStories.ControlledState} />

## Intent

Use `intent` prop to change the color of the checkbox.

<Canvas of={CheckboxStories.Intent} />

## Icon

Use `icon` prop to change the default checked icon.

<Canvas of={CheckboxStories.Icon} />

## Group

A checkbox group allows users to select one or more items from a list of choices.

### Usage

<Canvas of={CheckboxStories.Group} />

### Label

Checkbox groups should always have a label.

<Canvas of={CheckboxStories.GroupLabel} />

### Hidden Label

Similarly to the checkbox, in certain cases, a visible label may not be necessary, and for that, the `VisuallyHidden` component can be used.

<Canvas of={CheckboxStories.GroupHiddenLabel} />

### Uncontrolled

The `defaultValue` prop can be used to set the default state of the group. Optionally `onChange` prop can be used to get the current state of the checkbox group.

<Canvas of={CheckboxStories.GroupControlled} />

### Controlled

Use `value` and `onChange` props to control the state of the checkbox.

<Canvas of={CheckboxStories.GroupControlled} />

### Required

Use the `isRequired` prop to indicate that the checkbox group is required.

<Canvas of={CheckboxStories.GroupRequired} />

### Validation

Use the `isInvalid` prop to indicate that the checkbox group is invalid.

<Canvas of={CheckboxStories.GroupValidation} />

#### Individual validation

If an individual checkbox is invalid, wrap it with a `FormField` component a set the `isInvalid` prop to `true`.

<Canvas of={CheckboxStories.GroupIndividualValidation} />
